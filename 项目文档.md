# 智慧酒店预订 - 酒店比价平台项目文档

## 项目概述

本项目是一个酒店比价平台，参考trivago等酒店比价网站，针对国内酒店数据进行了优化。

## 核心功能

1. **酒店搜索**：目的地输入、日期选择、人数/客房设置、300+城市联想、搜索选项切换（按城市/按酒店）
2. **搜索结果**：酒店列表展示、图片、价格、评分、距离、多平台价格比较、排序功能（价格、评分、距离）
3. **酒店详情**：酒店详细信息、多平台价格展示、位置信息
4. **酒店比较**：多选比较功能，支持选择2家及以上酒店进行比较
5. **向导功能**：创意灯泡按钮引导，分步骤解说，自动滚动定位
6. **用户认证**：注册/登录/退出，session管理
7. **搜索历史**：保存/查看/快速填充，最多10条，支持按类型（城市/酒店）分类
8. **酒店收藏**：收藏/取消收藏/查看收藏
9. **热门搜索统计**：基于搜索历史的热门城市和酒店统计
10. **国际化支持**：多语言切换（中文、英文）
11. **响应式设计**：适配不同设备屏幕尺寸

## 数据库设计

### 表结构

1. **users表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - username: VARCHAR(50) UNIQUE NOT NULL
   - password: VARCHAR(50) NOT NULL
   - created_at: TIMESTAMP DEFAULT CURRENT_TIMESTAMP

2. **sessions表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - session_id: VARCHAR(50) UNIQUE NOT NULL
   - username: VARCHAR(50) NOT NULL
   - created_at: TIMESTAMP DEFAULT CURRENT_TIMESTAMP
   - FOREIGN KEY (username) REFERENCES users(username) ON DELETE CASCADE

3. **search_history表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - username: VARCHAR(50) NOT NULL
   - destination: VARCHAR(100) NOT NULL
   - city: VARCHAR(50)
   - hotel: VARCHAR(100)
   - type: VARCHAR(10) NOT NULL DEFAULT 'city'
   - check_in: DATE NOT NULL
   - check_out: DATE NOT NULL
   - timestamp: TIMESTAMP DEFAULT CURRENT_TIMESTAMP
   - FOREIGN KEY (username) REFERENCES users(username) ON DELETE CASCADE

4. **favorites表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - username: VARCHAR(50) NOT NULL
   - hotel_id: VARCHAR(50) NOT NULL
   - hotel_name: VARCHAR(100) NOT NULL
   - hotel_address: VARCHAR(200) NOT NULL
   - hotel_price: INT NOT NULL
   - hotel_image: VARCHAR(500) NOT NULL
   - created_at: TIMESTAMP DEFAULT CURRENT_TIMESTAMP
   - UNIQUE KEY unique_user_hotel (username, hotel_id)
   - FOREIGN KEY (username) REFERENCES users(username) ON DELETE CASCADE

5. **hotel_info表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - hotel_id: VARCHAR(50) UNIQUE NOT NULL
   - chn_name3: VARCHAR(100) NOT NULL
   - province: VARCHAR(50)
   - city_name: VARCHAR(50)
   - chn_address: VARCHAR(200)
   - pet_text: VARCHAR(200)
   - lng_baidu: VARCHAR(20)
   - lat_baidu: VARCHAR(20)

6. **price_detail表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - hotel_id: VARCHAR(50) NOT NULL
   - platform: VARCHAR(50) NOT NULL
   - price: DECIMAL(10,2) NOT NULL
   - distance: DECIMAL(10,2) NOT NULL
   - rating: DECIMAL(3,1) NOT NULL

7. **city表**
   - city_id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - province_id: INT NOT NULL
   - city_name: VARCHAR(50) NOT NULL
   - city_latitude: DECIMAL(10,6) NOT NULL
   - city_longitude: DECIMAL(10,6) NOT NULL

8. **provinces表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - province_code: VARCHAR(10) UNIQUE NOT NULL
   - province_name: VARCHAR(50) NOT NULL
   - created_at: TIMESTAMP DEFAULT CURRENT_TIMESTAMP

### 数据说明
- **酒店数据**：导入了224,550条酒店数据，覆盖全国主要城市
- **省份数据**：包含43个省份/地区（34个国内省份+9个国外地区）
- **搜索历史**：支持按城市和酒店类型统计热门搜索

## API接口

### 用户认证
| 接口 | 方法 | 请求体 | 响应 |
|------|------|--------|------|
| /api/register | POST | {username, password} | {success, username, sessionId} |
| /api/login | POST | {username, password} | {success, username, sessionId} |
| /api/logout | POST | {username} | {success} |
| /api/check-login | GET | - | {loggedIn, username, sessionId} |

### 搜索相关
| 接口 | 方法 | 参数 | 响应 |
|------|------|------|------|
| /api/suggestions | GET | query | Array[城市] |
| /api/history | GET | - | {success, history} |
| /api/history | POST | {destination, city, hotel, type, check_in, check_out} | {success} |
| /api/search | GET | city, hotel, page, limit, sort | {success, data: {hotels, total, page, limit, totalPages}} |

### 收藏相关
| 接口 | 方法 | 参数 | 响应 |
|------|------|------|------|
| /api/favorites | GET | - | {success, favorites} |
| /api/favorites | POST | {id, name, address, price, image} | {success} |
| /api/favorites/:id | DELETE | - | {success} |

### 热门相关
| 接口 | 方法 | 参数 | 响应 |
|------|------|------|------|
| /api/hot-hotels | GET | limit | {success, hotels} |
| /api/hot-cities | GET | limit | {success, cities} |

### 省份城市数据
| 接口 | 方法 | 参数 | 响应 |

## 技术亮点

1. **数据库迁移**：从MySQL迁移到SQLite (sql.js)，实现零配置部署
2. **搜索功能**：实现城市和酒店搜索，支持热门搜索统计
3. **搜索结果缓存**：实现搜索结果缓存机制，提高响应速度
4. **历史记录**：无论登录与否都能统计搜索次数
5. **数据导入**：导入了22万+酒店数据和省份数据，包含完整的酒店信息和价格详情
6. **数据处理脚本**：提供了一系列数据处理脚本，用于数据清洗、导入和验证
7. **界面交互**：优化了用户界面和交互体验，包括响应式设计和动画效果
8. **模块化架构**：前端和后端都采用模块化设计，便于维护和扩展
9. **国际化支持**：支持多语言切换，包括中文和英文
10. **多平台价格比较**：在搜索结果中展示多家平台的价格，方便用户比较
