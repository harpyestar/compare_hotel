# 智慧酒店预订 - 酒店比价平台项目文档

## 项目概述

本项目是一个酒店比价平台，参考trivago等酒店比价网站，针对国内酒店数据进行了优化。

## 核心功能

1. **酒店搜索**：目的地输入、日期选择、人数/客房设置、300+城市联想
2. **热门酒店推荐**：酒店列表展示、图片、价格、评分
3. **酒店比较**：多选比较功能
4. **向导功能**：创意灯泡按钮引导，分步骤解说，自动滚动定位
5. **用户认证**：注册/登录/退出，session管理
6. **搜索历史**：保存/查看/快速填充，最多10条
7. **酒店收藏**：收藏/取消收藏/查看收藏
8. **热门搜索统计**：基于搜索历史的热门城市和酒店统计

## 数据库设计

### 表结构

1. **users表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - username: TEXT UNIQUE NOT NULL
   - password: TEXT NOT NULL
   - created_at: DATETIME DEFAULT CURRENT_TIMESTAMP

2. **sessions表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - session_id: TEXT UNIQUE NOT NULL
   - username: TEXT NOT NULL
   - created_at: DATETIME DEFAULT CURRENT_TIMESTAMP

3. **search_history表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - username: TEXT NOT NULL
   - destination: TEXT NOT NULL
   - city: TEXT
   - hotel: TEXT
   - type: TEXT NOT NULL DEFAULT 'city'
   - check_in: DATE NOT NULL
   - check_out: DATE NOT NULL
   - timestamp: DATETIME DEFAULT CURRENT_TIMESTAMP

4. **favorites表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - username: TEXT NOT NULL
   - hotel_id: TEXT NOT NULL
   - hotel_name: TEXT NOT NULL
   - hotel_address: TEXT NOT NULL
   - hotel_price: INTEGER NOT NULL
   - hotel_image: TEXT NOT NULL
   - created_at: DATETIME DEFAULT CURRENT_TIMESTAMP

5. **hotel_info表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - hotel_id: TEXT UNIQUE NOT NULL
   - chn_name3: TEXT NOT NULL
   - province: TEXT
   - city_name: TEXT
   - chn_address: TEXT
   - pet_text: TEXT
   - lng_baidu: TEXT
   - lat_baidu: TEXT

6. **provinces表**
   - id: INTEGER PRIMARY KEY AUTO_INCREMENT
   - province_code: TEXT UNIQUE NOT NULL
   - province_name: TEXT NOT NULL
   - created_at: DATETIME DEFAULT CURRENT_TIMESTAMP

### 数据说明
- **酒店数据**：导入了224,550条酒店数据，覆盖全国主要城市
- **省份数据**：包含43个省份/地区（34个国内省份+9个国外地区）
- **搜索历史**：支持按城市和酒店类型统计热门搜索

## API接口

### 用户认证
| 接口 | 方法 | 请求体 | 响应 |
|------|------|--------|------|
| /api/register | POST | {username, password} | {success, username, sessionId} |
| /api/login | POST | {username, password} | {success, username, sessionId} |
| /api/logout | POST | {username} | {success} |
| /api/check-login | GET | - | {isLogin, username} |

### 搜索相关
| 接口 | 方法 | 参数 | 响应 |
|------|------|------|------|
| /api/suggestions | GET | query | Array[城市] |
| /api/history | GET | - | Array[搜索记录] |
| /api/history | POST | {destination, city, hotel, type, check_in, check_out} | {success} |

### 收藏相关
| 接口 | 方法 | 参数 | 响应 |
|------|------|------|------|
| /api/favorites | GET | - | Array[收藏酒店] |
| /api/favorites | POST | {hotel_id, hotel_name, hotel_address, hotel_price, hotel_image} | {success} |
| /api/favorites/:id | DELETE | - | {success} |

### 热门相关
| 接口 | 方法 | 参数 | 响应 |
|------|------|------|------|
| /api/hot-hotels | GET | limit | Array[酒店] |
| /api/hot-cities | GET | limit | Array[城市] |

## 技术亮点

1. **数据库迁移**：从MySQL迁移到SQLite (sql.js)，实现零配置部署
2. **搜索功能**：实现城市和酒店搜索，支持热门搜索统计
3. **历史记录**：无论登录与否都能统计搜索次数
4. **数据导入**：导入了22万+酒店数据和省份数据
5. **界面交互**：优化了用户界面和交互体验
6. **模块化架构**：前端和后端都采用模块化设计，便于维护和扩展
7. **国际化支持**：支持多语言切换
