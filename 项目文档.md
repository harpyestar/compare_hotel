# 干就完了 - 酒店比价平台项目文档

## 项目概述

本项目是一个酒店比价平台的前端实现，参考trivago等酒店比价网站，针对国内酒店数据进行了优化。

## 项目结构

```
├── index.html                      # 主页面
├── css/
│   ├── base.css                    # 基础样式
│   ├── style.css                   # 主样式
│   └── components/                 # 组件样式
│       ├── header.css, search.css, hotel.css, modal.css, guide.css, footer.css
├── js/
│   ├── main.js                     # 主入口
│   └── modules/                    # 功能模块
│       ├── core.js, search.js, datepicker.js, guests.js
│       ├── user.js, favorites.js, guide.js
├── components/                      # HTML组件
│   ├── header.html, search.html, modal.html, footer.html
├── server/server.js                # 后端服务器
└── 项目文档.md
```

## 核心功能

1. **酒店搜索**：目的地输入、日期选择、人数/客房设置、300+城市联想
2. **热门酒店推荐**：酒店列表展示、图片、价格、评分
3. **酒店比较**：多选比较功能
4. **向导功能**：创意灯泡按钮引导，分步骤解说，自动滚动定位
5. **用户认证**：注册/登录/退出，session管理
6. **搜索历史**：保存/查看/快速填充，最多10条
7. **酒店收藏**：收藏/取消收藏/查看收藏

## 技术栈

- **前端**：HTML5, CSS3, JavaScript, Bootstrap 5, Font Awesome
- **后端**：Node.js, Express, MySQL

## 关键API

| 接口 | 方法 | 功能 |
|------|------|------|
| /api/register | POST | 用户注册 |
| /api/login | POST | 用户登录 |
| /api/logout | POST | 退出登录 |
| /api/check-login | GET | 检查登录状态 |
| /api/history | GET/POST | 搜索历史 |
| /api/favorites | GET/POST | 收藏酒店 |
| /api/favorites/:id | DELETE | 取消收藏 |
| /api/suggestions | GET | 城市联想 |

## 启动服务器

```bash
npm start
```

访问地址：http://localhost:3000
