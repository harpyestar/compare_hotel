# 干就完了 - 酒店比价平台项目文档

## 项目概述

本项目是一个酒店比价平台的前端实现，旨在为用户提供方便的酒店搜索和比较功能。平台参考了 trivago 等酒店比价网站的设计理念，针对国内酒店数据进行了优化。

## 核心功能

### 1. 酒店搜索功能
- 目的地输入框，支持城市和酒店名称搜索
- 入住和离店日期选择
- 热门目的地向导，点击即可快速填充
- **搜索联想功能**：输入城市名称时，自动显示匹配的城市列表，支持滚动浏览

### 2. 合作平台展示
- 展示与多家国内酒店预订平台的合作关系
- 平台logo和名称展示（携程、美团、飞猪、艺龙）

### 3. 热门酒店推荐
- 展示国内热门酒店列表
- 包含酒店名称、地址、价格、评分等信息
- 酒店图片展示
- 热门搜索向导，点击即可快速搜索

### 4. 酒店比较功能
- 支持选择多个酒店进行比较
- 比较按钮，显示已选择的酒店数量
- 点击比较按钮可查看比较结果

### 5. 向导功能
- **创意灯泡按钮**：位于页面右上角，采用灯泡造型设计，带有光芒效果
- 点击后弹出向导弹窗，背景变暗
- 分步骤引导用户使用平台功能（搜索栏、合作平台、热门酒店）
- 高亮显示当前介绍的区域
- 支持上一步/下一步导航
- 自动滚动到目标区域，确保高亮区域在视口中可见

### 6. 用户认证功能
- **用户注册**：支持新用户注册，创建账号
- **用户登录**：支持用户登录，保存登录状态
- **退出登录**：支持用户退出登录
- **会话管理**：使用localStorage保存会话ID，确保登录状态持久化

### 7. 搜索历史功能
- **保存搜索历史**：自动保存用户的搜索记录
- **查看搜索历史**：用户可以查看历史搜索记录
- **快速填充**：点击历史记录可快速填充搜索表单
- **历史记录限制**：最多保存10条搜索历史

### 8. 酒店收藏功能
- **收藏酒店**：用户可以收藏喜欢的酒店
- **查看收藏**：用户可以查看已收藏的酒店
- **取消收藏**：用户可以取消收藏酒店
- **收藏状态显示**：收藏后按钮图标会发生变化

## 技术实现

### 前端技术栈
- HTML5：页面结构
- CSS3：样式设计
- JavaScript：交互功能
- Bootstrap 5：响应式布局
- Font Awesome：图标库

### 后端技术栈
- Node.js：运行环境
- Express：Web框架
- CORS：跨域资源共享

### 关键API实现

#### 用户认证API
- `/api/register`：用户注册
- `/api/login`：用户登录
- `/api/logout`：退出登录
- `/api/check-login`：检查登录状态

#### 搜索历史API
- `/api/history`（POST）：保存搜索历史
- `/api/history`（GET）：获取搜索历史

#### 酒店收藏API
- `/api/favorites`（POST）：收藏酒店
- `/api/favorites`（GET）：获取收藏酒店
- `/api/favorites/:id`（DELETE）：取消收藏酒店

### 关键组件实现

#### 搜索框组件
- 使用 Bootstrap 表单组件
- 响应式设计，适配不同屏幕尺寸
- 集成热门目的地向导
- **搜索联想功能**：
  - 前端：监听输入事件，发送请求到后端获取联想结果
  - 后端：提供 `/api/suggestions` 接口，返回匹配的城市列表
  - 支持滚动浏览联想结果

#### 酒店卡片组件
- 包含酒店图片、名称、地址、价格、评分等信息
- 悬停效果，提升用户体验
- 比较复选框，支持选择多个酒店

#### 向导组件
- 模态弹窗设计
- 步骤切换功能（上一步/下一步）
- 目标元素高亮效果
- 背景遮罩，突出弹窗内容
- 自动滚动到目标区域
- 平滑过渡动画
- **创意灯泡按钮**：
  - 采用灯泡造型设计，带有黄色光芒效果
  - 悬停时放大并显示更明显的阴影效果
  - 位于页面右上角，固定定位

## 数据处理

目前项目使用占位数据进行展示，包括：
- 酒店图片：使用在线生成的图片
- 酒店信息：模拟数据
- 平台logo：使用在线生成的图片
- **城市数据**：包含全国所有主要城市（300+个城市），用于搜索联想功能

后续可通过API获取真实数据进行替换。

## 使用说明

### 启动服务器

1. **打开终端**：在项目根目录 `d:\公司文件夹\房仓\A+\Once` 中打开终端
2. **启动服务器**：运行以下命令
   ```bash
   npm start
   ```
3. **服务器状态**：当看到以下信息时，表示服务器启动成功
   ```
   Server running on http://localhost:3000
   数据库初始化完成
   数据库连接成功，使用MySQL存储数据
   ```

### 访问平台

1. **打开浏览器**：在浏览器中输入以下地址
   ```
   http://localhost:3000
   ```
2. **使用功能**：
   - **搜索酒店**：在搜索框中输入目的地，选择入住和离店日期，点击搜索按钮。
   - **搜索联想**：输入城市名称时，会自动显示匹配的城市列表，可滚动浏览并点击选择。
   - **快速搜索**：点击热门目的地或热门搜索标签，快速填充搜索条件。
   - **浏览酒店**：在热门酒店推荐区域浏览酒店列表。
   - **比较酒店**：勾选酒店卡片上的复选框，选择多个酒店后点击右下角的比较按钮。
   - **使用向导**：点击右上角的灯泡按钮，按照向导步骤了解平台功能，可使用上一步/下一步按钮导航。
   - **用户注册**：点击页面顶部的"注册"按钮，填写用户名和密码，创建新账号。
   - **用户登录**：点击页面顶部的"登录"按钮，输入用户名和密码，登录账号。
   - **查看搜索历史**：登录后，点击"搜索历史"按钮，查看历史搜索记录，点击记录可快速填充搜索表单。
   - **收藏酒店**：在酒店卡片上点击心形图标，收藏喜欢的酒店。
   - **查看收藏**：登录后，点击"我的收藏"按钮，查看已收藏的酒店。
   - **退出登录**：点击"退出登录"按钮，退出当前账号。
